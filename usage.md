# HealthMate - Doctor & Lab Booking App

HealthMate is a premium healthcare platform inspired by the Zomato UI/UX, designed to make booking doctor appointments and diagnostic tests as easy as ordering food.

## ✨ Key Features
- **Zomato-inspired UI**: Clean, modern, and highly aesthetic interface.
- **Dual Flow**: Seamlessly switch between **Doctor Appointments** and **Diagnostic Labs**.
- **Location-based Search**: Targeted search for healthcare providers in specific areas.
- **Smart Filters**: Filter doctors by specialty (Cardiology, Pediatrics, etc.).
- **Interactive Booking**: A smooth booking modal with time-slot selection and instant confirmation.
- **Responsive Design**: Fully functional on mobile and desktop devices.

## 👥 User Roles & Dashboards

The app now supports four distinct roles. Use the **Role Switcher** in the top right to toggle between them:

### 1. PATIENT (Default)
- **Features**: Zomato-style search, high-quality listings, and interactive booking.
- **Goal**: Find and book healthcare services quickly.

### 2. DOCTOR
- **Features**: Dashboard showing today's appointment count, earnings summary, and an interactive patient list.
- **Actions**: Start consultations or reschedule appointments directly from the list.

### 3. LAB
- **Features**: List of test requests, sample tracking, and quick stats for pending tests.
- **Actions**: Update test statuses (e.g., "Sample Collected") and manage new test catalog items.

### 4. ADMIN
- **Features**: Platform-wide management interface for verifying new doctors and labs.
- **Actions**: Review submitted medical licenses or NABL certifications and Approve/Reject providers.
